# Contributing to GitHub Hub 🤝

We would love for you to contribute to our project and help make it even better! As a contributor, here are the guidelines we would like you to follow:

## 📜 Code of Conduct

Help us keep GitHub Hub open and inclusive. Please read and follow our Code of Conduct (be kind, respectful, and professional to all community members).

## 💡 How Can I Contribute?

### 🐛 Reporting Bugs
If you find a bug in the source code, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.

### 🌟 Suggesting Enhancements
If you have an idea for a new feature or improvement, please open a feature-request issue! We value our community's input.

### 💻 Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Run `pnpm install`** to install dependencies.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes (`pnpm test` or `pnpm lint`).
5. Make sure your code lints.
6. Commit your changes using descriptive commit messages.
7. Issue that pull request!

## 🧑‍💻 Development Setup

The project is built using:
- **React 18** (Vite)
- **Tailwind CSS v4** (Utility-first styling, glassmorphism)
- **Framer Motion** (Animations and transitions)
- **Lucide React** (Beautiful iconography)
- **Shadcn UI & Radix UI** (Accessible, unstyled primitives)

All application code lives in `apps/frontend/src`. Ensure you use `pnpm` as the package manager to respect the `pnpm-workspace.yaml` architecture.
